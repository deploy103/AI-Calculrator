# Level 4 AI Agent 계산기 구상

## 목표

사용자가 자연어로 질문하면 AI가 의도를 파악해서 계산 계획을 세우고, 실제 계산은 파이썬이 안전하게 수행하는 계산기를 만든다.

예시:

- "18명의 학생이 3명씩 팀을 만들면 몇 팀이야?"
- "원 반지름이 5일 때 넓이 구해줘"
- "sin(pi / 2) 값은?"
- "12000원의 15% 할인가는 얼마야?"

## 구조

1. 사용자가 자연어로 질문한다.
2. OpenAI 모델이 질문을 읽고 계산 계획을 JSON 형태로 만든다.
3. 파이썬 프로그램이 그 JSON에서 수식만 꺼낸다.
4. 파이썬이 허용된 연산만 안전하게 계산한다.
5. 계산 결과와 함께 해석 문장을 출력한다.

## 왜 이렇게 만드나

- AI가 자연어 이해를 맡는다.
- 파이썬이 실제 계산을 맡아서 결과를 통제할 수 있다.
- 프롬프트와 계산 로직을 분리할 수 있다.
- 나중에 GUI 버전이나 음성 버전으로 확장하기 쉽다.

## 구현 범위

### 1차 구현

- Tkinter GUI 기반 AI Agent 계산기
- OpenAI Responses API 사용
- 기본 모델은 `gpt-5.4`
- 사칙연산, 괄호, 거듭제곱, 나머지, 소수 계산
- `sin`, `cos`, `tan`, `sqrt`, `log10`, `ln`, `pi`, `e` 지원
- 사용자의 질문을 AI가 `expression`, `formula_text`, `explanation`, `answer_unit` 형태로 정리
- 결과 화면에서 `식`, `설명`, `결과`를 함께 보여줌

### 2차 확장

- 계산 기록 저장하기
- 단위 변환 추가하기
- 문제 유형별 프롬프트 분기하기
- 여러 단계 풀이를 보여주기

## 파일 계획

- `ai_agent_calculator.py`: 메인 실행 파일
- `requirements.txt`: 필요한 패키지 목록

## 실행 방식

1. `OPENAI_API_KEY` 환경 변수를 등록하거나 `.env/.env` 파일에 키를 넣는다.
2. `python3 ai_agent_calculator.py` 실행
3. GUI 창에서 질문을 입력하고 결과를 확인한다

## 핵심 차별점

- 일반 계산기처럼 버튼만 누르는 게 아니라 자연어를 이해한다.
- 답만 주는 것이 아니라 왜 그 답이 나왔는지 식과 설명을 같이 보여준다.
- "AI가 해석, 파이썬이 계산" 구조라서 설명하기 좋다.
- Level 1, 2, 3에서 만든 계산기보다 한 단계 위 구조로 보인다.

## 주의

- API 키가 없으면 실행되지 않는다.
- AI가 이상한 수식을 만들 수 있으므로 파이썬 쪽에서 허용된 연산만 실행해야 한다.
- 공식 OpenAI 문서 기준으로 새 프로젝트에는 Responses API를 사용하는 방향이 적절하다.
